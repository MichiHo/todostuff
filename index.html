About

This program operates on a todo.txt file, as introduced by Gina Trapani. In this single file, each line corresponds to a task, which can contain a single-letter priority, several projects and resources and also arbitrary properties, and is marked as done by preceding it with an x.

x (A) Save the world +lifegoals @superpowers due:2020-09-02

The main file could be stored in the cloud to enable easy synchronization with other computers or various mobile apps already available for todo.txt.

Panels

With this app you can view your todo.txt-file filtered in different ways, each displayed as a panel. The filters are configured with entered text similar to the entries themselves, with some additional options. Priority, projects and resources are automatically set for new entries created in a panel.

Filters
  • (A) only prio A items
    (<A) items with prio lower than A, including items without prio
    (>C) only prio A and B items
  • Only items with +thisProject and @thisResource are shown
  • Items with !+thisProject and !@thisResource are hidden
  • %done only completed tasks
    !x only uncompleted tasks
  • %remains shows only tasks not selected by any panel with filters and without %remains
  • Any other entered text can be used to only show tasks containing this text
